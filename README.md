# 北京千面科技有限公司 官方网站

> 让科技更有温度

## 项目介绍

这是北京千面科技有限公司的官方网站，采用纯 HTML + CSS + JavaScript 技术栈构建的单页应用（SPA）网站。

## 技术栈

- **HTML5**: 语义化 Markup
- **CSS3**: 响应式设计、动画效果、Flex/Grid 布局
- **JavaScript (ES6+)**: 原生 JavaScript，无框架依赖
- **字体**: Google Fonts - Noto Sans SC

## 浏览器支持

- Chrome (最新)
- Firefox (最新)
- Safari (最新)
- Edge (最新)

## 项目结构

```
hippo-website/
├── index.html      # 主页面
├── css/
│   └── style.css  # 样式文件
├── js/
│   └── main.js    # 交互脚本
├── images/         # 图片资源（预留）
└── README.md     # 说明文档
```

## 功能特性

### 页面模块
1. **导航栏** - 固定顶部，平滑滚动，响应式适配
2. **Hero 区域** - 大屏展示，粒子动画效果
3. **关于我们** - 公司简介，统计数据，企业愿景
4. **业务范围** - 6大核心业务卡片展示
5. **解决方案** - 行业解决方案标签页切换
6. **合作伙伴** - 合作伙伴展示
7. **联系我们** - 联系信息 + 在线留言表单
8. **页脚** - 底部导航，快速链接

### 交互功能
- 滚动导航高亮
- 移动端菜单
- 解决方案标签页切换
- 联系表单验证与提交
- 滚动 Reveal 动画
- 粒子背景效果

## 本地运行

### 方式一：直接打开
双击 `index.html` 在浏览器中打开

### 方式二：使用简单服务器
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .
```

然后访问 http://localhost:8000

## 自定义修改

### 修改公司信息
编辑 `index.html` 中的以下内容：
- 公司地址（第 XXX 行）
- 联系方式（第 XXX 行）
- 公司简介（第 XXX 行）

### 修改配色
编辑 `css/style.css` 中的 CSS 变量：
```css
:root {
    --primary-color: #0066ff;    /* 主色调 */
    --secondary-color: #00ccaa;   /* 辅色调 */
}
```

### 添加实际图片
将公司实际图片放入 `images/` 目录，然后在 HTML 中替换占位图。

## 注意事项

1. 本网站使用了 Google Fonts，需要确保网络环境可以访问 fonts.googleapis.com
2. 联系表单为前端演示，实际使用需要对接后端服务
3. 网站为静态页面，如需SEO优化建议添加服务端渲染
